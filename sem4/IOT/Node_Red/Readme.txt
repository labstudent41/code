Check if node red is installed by running the node red server.
Try any of these commands to start node red :-
    node-red-start
    node-red-pi
    node-red

After the server starts, an http link is provided which looks like :-
    http://127.0.0.1:1880/
Click on the link and wait until the browser opens.
If it doesn't, manually copy and paste it in browser.

From the left panel, get rgpio-in, rgpio-out, etc. and connect them to build a circuit
